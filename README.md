# w8loss

## Initialize
1. `git clone git@github.com:n8io/w8loss.git && cd w8loss`
2. `npm install`

## To build dev assets
`grunt dev`

## To build prod assets
`grunt`

## To run node app
`npm start`

## To watch assets for changes, rebuild, and start the app
`grunt watcher`

## To view app
[http://localhost:3000](http://localhost:3000)
